# Sistemas Operacionais: Guia de Conteúdo e Segurança

## 1. Interface de Linha de Comando (CLI) do Windows

### Navegação e Pastas
* Comando `dir` para listagem de arquivos e diretórios atuais.
* Comando `cd` para navegação entre as pastas do sistema.
* Comando `mkdir` para criação de novos diretórios pelo terminal.

### Gerenciamento de Usuários e Variáveis
* Comando `net user` para gerenciar contas de usuários locais.
* Comando `set` para visualizar e configurar variáveis de ambiente.
* Variável `%PATH%` para armazenamento de caminhos de executáveis globais.

---

## 2. Interface de Linha de Comando (CLI) do Linux

### Comandos Essenciais de Arquivos
* Comando `ls` para listar o conteúdo do diretório atual.
* Comando `cd` para alternar entre as pastas do sistema.
* Comando `mkdir` para criar novos diretórios na estrutura de arquivos.

### Permissões e Usuários
* Comando `chmod` para modificar as permissões de acesso a arquivos.
* Comando `chown` para alterar o proprietário de arquivos e diretórios.
* Comando `sudo` para executar tarefas com privilégios de superusuário.

---

## 3. Distribuições Linux

### Ecossistemas Principais
* Família Debian: Foco em estabilidade, base para o Ubuntu.
* Família Red Hat (RHEL): Foco corporativo, base para o Fedora.
* Alpine Linux: Distribuição ultra-leve focada no uso em containers.

### Gerenciadores de Pacotes
* Ferramenta `apt` para instalação de softwares em sistemas Debian/Ubuntu.
* Ferramenta `dnf` para gerenciamento de pacotes no ecossistema Red Hat.
* Atualização do sistema e resolução automática de dependências de código.

---

## 4. Fundamentos de Segurança Cibernética

### Segurança no Sistema Operacional
* Princípio do Privilégio Mínimo: Limitação de acessos por usuário.
* Auditoria de logs do sistema para identificação de invasões.
* Desativação de serviços e portas de rede não utilizados.

### Ameaças Comuns e Mitigação
* Malware: Códigos maliciosos executados sem o consentimento do usuário.
* Atualizações de segurança (patches) para correção de vulnerabilidades críticas.
* Configuração de Firewalls locais para controle de tráfego de rede.
