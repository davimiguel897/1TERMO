# Lógica de Programação com Python: Guia de Conteúdo e Boas Práticas

## 1. Conteúdo de Aula: Fundamentos de Python

### Variáveis e Tipos de Dados
* Armazenamento de informações na memória do computador.
* Tipos primitivos básicos: `int`, `float`, `str` e `bool`.
* Tipos de coleções de dados: `list`, `tuple` e `dict`.

### Estruturas de Controle
* Estruturas condicionais para tomada de decisão: `if`, `elif` e `else`.
* Laços de repetição para automação de tarefas: `while` e `for`.
* Controle de fluxo em coleções com o uso de `range()`.

---

## 2. Controle de Versão: Git e GitHub

### Git (Local)
* Sistema distribuído para controle de histórico de código.
* Comando `git init` para iniciar o monitoramento do projeto.
* Comando `git add` para preparar arquivos para o salvamento.
* Comando `git commit` para gravar as alterações localmente.

### GitHub (Remoto)
* Plataforma em nuvem para hospedagem de repositórios Git.
* Comando `git push` para enviar o código local para a nuvem.
* Ferramenta essencial para o trabalho colaborativo em software.
* Portfólio público para demonstração de projetos de código.

---

## 3. Qualidade de Código: Clean Code em Python

### Regras de Ouro
* Nomes de variáveis devem ser significativos e legíveis.
* Funções devem ser pequenas e realizar apenas uma tarefa.
* Código limpo dispensa o excesso de comentários explicativos.

### Padrão PEP 8
* Guia oficial de estilo para escrita de código Python.
* Uso de 4 espaços por nível de indentação vertical.
* Padrão `snake_case` para nomes de variáveis e funções.
